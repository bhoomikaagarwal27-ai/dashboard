# 📊 Rajsha Chemicals Dashboard

A real-time industrial dashboard developed during my Data Analyst Internship at **Rajsha Chemicals Pvt Ltd (TWC Group)**.

---

## 🚀 Project Overview

This system enables daily operational data tracking across multiple departments and provides real-time insights to senior management.

---

## 🛠️ Tech Stack

- Frontend: HTML, CSS, JavaScript  
- Backend: Google Apps Script  
- Database: Google Sheets  
- Hosting: GitHub Pages  

---

## 🔑 Key Features

- 🔐 Department-based login system  
- 📥 Daily data entry by staff  
- 🔄 Automatic sync with Google Sheets  
- 📊 Real-time dashboard for senior management  
- 🚨 Alert system (plan vs actual deviations)  
- 📱 WhatsApp-ready reporting feature  

---

## 🏭 Departments Covered

- Production  
- Dispatch  
- Electricity  
- Water Consumption  
- Safety  
- HR / Manpower  

---

## 📈 Key Functionalities

- KPI Calculation (Efficiency, Output, etc.)
- Department-wise performance tracking
- Automated reporting system
- Live data visualization

---

## 🧠 Business Impact

- Reduced manual reporting effort  
- Improved decision-making speed  
- Centralized data system  
- Increased operational transparency
